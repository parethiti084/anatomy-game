# anatomy-game

## Anatomy Learning Game

An interactive Anatomy Learning Game developed using Flask.

## Features

- Multiple-choice anatomy quiz
- Automatic score calculation
- Dashboard showing:
  - Total Score
  - Correct Answers
  - Incorrect Answers
  - Accuracy Percentage
- Responsive interface

## Technology

- Python
- Flask
- HTML
- CSS
- JavaScript
